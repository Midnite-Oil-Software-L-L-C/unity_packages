# Core Components for Midnite Oil Software Projects
## Installation Options

### Option 1: Install via Unity Package Manager (Recommended)
1. Open Unity and go to `Window > Package Manager`.
2. Click the "+" button and choose `Add package from git URL`.
3. Enter the following URL: https://github.com/Midnite-Oil-Software-L-L-C/unity_packages.git?path=/Core

### Option 2: Download and Import `.unitypackage`
1. Go to the [Releases](https://github.com/Midnite-Oil-Software-L-L-C/unity_packages/releases) section of this repository.
2. Download the latest `.unitypackage` file.
3. Drag and drop the file into Unity, or import it via `Assets > Import Package > Custom Package...`.
